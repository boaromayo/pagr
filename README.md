### About

This repository stores a copy of the scripts for the 2006 [RPG Advocate](http://rpgmaker.wikia.com/wiki/RPG_Advocate) game, **Phylomortis: Avant Garde**.

The game scripts are coded in Ruby, and are used for the game engine RPG Maker XP.

### Overview

Phylomortis: Avant-Garde's plot is complex, with the reading level of the dialogue at graduate school-reading level.

More information about the game could be found at [the archived official site](https://web.archive.org/web/20080131101636/http://www.phylomortis.com:80/html/agstory.html).
