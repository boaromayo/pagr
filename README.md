## About

This repository stores a copy of the scripts for the 2006 [RPG Advocate](https://web.archive.org/web/20080131101636/http://www.phylomortis.com:80/html/agstory.html) game, **Phylomortis: Avant Garde**.

The game scripts are coded in Ruby, and are used for the game engine RPG Maker XP.
